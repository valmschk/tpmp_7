# Задание 2. Графические объекты на Swift (MVC + булевы операции)

**Вариант 32**  
**Выполнила:** Мещенко Валерия, 2 курс 10 группа

## Реализованные фигуры

### 1. Равносторонний треугольник
- Координаты вершин рассчитаны математически
- Высота = (√3/2) × сторона
- Центрирован на экране

### 2. Октаграмма (8-конечная звезда)
- 16 точек (чередование внешнего/внутреннего радиуса)
- Внешний радиус: 120 pt
- Внутренний радиус: 50 pt

## Булевы операции (главная часть задания)

| Режим | Описание | Реализация |
|-------|----------|-------------|
| **Раздельное отображение** | Треугольник и звезда в разных частях экрана | Два независимых `CAShapeLayer` |
| **Вычитание** | Звезда вырезана из треугольника | `fillRule = .evenOdd` |
| **Объединение** | Одна сложная фигура | Общий путь + градиент |

## Визуальные эффекты

-  **Градиент** (`CAGradientLayer`) + маска (`mask`)
-  **Тень** (`shadowColor`, `shadowOpacity`, `shadowRadius`)
- **Плавное переключение** между режимами по тапу
